# 🚀 Mini CRM System

A full-stack CRM application to manage leads, built using HTML, CSS, JavaScript, Node.js, Express, and MongoDB.

---

## 🔗 Live Demo

🌐 Frontend (Netlify):  
https://future-mini-crm.netlify.app

⚙️ Backend (Render):  
https://future-fs-02-95jz.onrender.com

---

## ✨ Features

- User Registration & Login (JWT Authentication)
- Add / Update / Delete Leads
- Lead Status Tracking (New / Contacted / Converted)
- Notes for each lead
- Dashboard with analytics chart
- Protected routes (login required)

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Hosting: Netlify + Render
