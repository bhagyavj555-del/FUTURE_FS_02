# Mini CRM Project

This is a simple CRM (Customer Relationship Management) web application.

## Features
- User Registration & Login
- Add / Delete Leads
- Dashboard with charts
- Modern UI with sidebar

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## How to Run
1. Install dependencies:
   npm install

2. Start backend:
   node server.js

3. Open frontend in browser
